## Español

Panel de control administrativo avanzado para la gestión de personal y asistencia, diseñado para centralizar la información recolectada por el ecosistema StaffAxis.

### 🚀 Características
- **Supervisión de Sectores**: Monitoreo en tiempo real de múltiples áreas operativas (Hortalizas, Granos, Industrial, etc.).
- **Estadísticas de Personal**: Visualización de empleados registrados, activos y ausentes por período.
- **Exportación Avanzada**: Generación de reportes de asistencia en Excel con filtros por sector y períodos personalizados (21 al 20).
- **Gestión de RRHH**: Alta, baja y modificación de empleados y sectores desde una interfaz centralizada.
- **Visualización de Datos**: Gráficos interactivos y tendencias de asistencia.
- **Tecnología de Escritorio**: Aplicación robusta basada en Electron para entornos Windows.

### 🛠️ Stack Tecnológico
- **Frontend**: React + TypeScript
- **Estilos**: Tailwind CSS + Material UI + Radix UI
- **Desktop**: Electron
- **Gráficos**: Recharts
- **Utilidades**: ExcelJS / XLSX para reportes complejos

### 📂 Estructura del Proyecto
- `src/app/`: Lógica principal de la aplicación y componentes de UI.
- `electron/`: Configuración del proceso principal de la ventana nativa.
- `dist-electron-bin/`: Binarios y ejecutables finales de la aplicación.

## English

An advanced administrative dashboard for personnel and attendance management, designed to centralize information collected by the StaffAxis ecosystem.

### 🚀 Key Features
- **Sector Supervision**: Real-time monitoring of multiple operational areas (Vegetables, Grains, Industrial, etc.).
- **Personnel Statistics**: Visualization of registered, active, and absent employees per period.
- **Advanced Exporting**: Generation of Excel attendance reports with filters by sector and custom periods (21st to 20th).
- **HR Management**: Create, update, and delete employees and sectors from a centralized interface.
- **Data Visualization**: Interactive charts and attendance trends.
- **Desktop Technology**: Robust Electron-based application for Windows environments.

### 🛠️ Tech Stack
- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS + Material UI + Radix UI
- **Desktop Framework**: Electron
- **Charts**: Recharts
- **Utilities**: ExcelJS / XLSX for complex reporting

### 📂 Project Structure
- `src/app/`: Main application logic and UI components.
- `electron/`: Configuration for the native window main process.
- `dist-electron-bin/`: Application binaries and final executables.
